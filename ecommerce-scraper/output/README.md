# E-Commerce Product Scraper using Playwright

This project is a **professional web scraper** built with **Playwright** in JavaScript.  
It scrapes product data from an e-commerce website (Books to Scrape) and exports it into a CSV file.  

---

## Features

- Scrapes product details including:
  - **Title**
  - **Price**
  - **Rating**
  - **Product URL**
- Handles **pagination** automatically
- Saves data to **CSV** for easy analysis
- Built with **Playwright** for JavaScript-rendered websites
- Fully **portfolio and client-ready**  

---

## Tech Stack

- Node.js  
- Playwright  
- JavaScript  
- CSV file output
