# Job Scraper using Playwright

This project is a job listings scraper built with **Playwright (JavaScript)**.
It demonstrates real-world web scraping concepts including DOM extraction,
pagination handling, and exporting data to CSV format.

## Features
- Scrapes job title, company, location, and link
- Handles pagination logic
- Saves extracted data to a CSV file
- Built with Playwright for JavaScript-rendered websites

## Tech Stack
- Node.js
- Playwright
- JavaScript
